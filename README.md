🍷 TGWine (Telegram Giveaway Winner)
Автоматизированный инструмент для участия в розыгрышах Telegram (Giveaways) с использованием нескольких аккаунтов (Userbots).

🚀 Основные возможности
Многопоточность: Работа с неограниченным количеством сессий Telegram.

Автоматизация: Бот сам находит сообщения с розыгрышами, подписывается на необходимые каналы и нажимает на кнопку участия (обычно «Участвовать» или 🎉).

Поддержка сессий: Работает с файлами .session (библиотека Pyrogram).

Обход проверок: Имитация действий реального пользователя для минимизации риска блокировки.

🛠 Стек технологий
Язык: Python 3.10+

Хранилище: SQLite (для управления сессиями и логами)
